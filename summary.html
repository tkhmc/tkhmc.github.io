マイコン班の音

ビープ音

解説

マイコン班の音。 PCの内部にある小さなスピーカーから発せられる音のこと。
例えば、C#ではConsole.Beep()を使うことで鳴らすことができる。
マイコン班では音にこれを使うことを推奨している…はず。

サンプル

サンプルプロジェクト

注意

Windows 95では周波数と時間は無視されます。

はもともと対応してないけど。

プロトタイプ

Win32API C/C++用

BOOL Beep(
  DWORD dwFreq,      // 音の周波数
  DWORD dwDuration   // 音の持続時間
);

標準Cライブラリ

printf("\a");

対応情報

Windows NT/2000:Windows NT 3.1 以降
Windows 95/98:Windows 95 以降
ヘッダー:Winbase.h 内で宣言、Windows.h をインクルード
インポートライブラリ:Kernel32.lib を使用
