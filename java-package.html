Javaパッケージ名

概要

マイコン班は東海中学校・東海高校のJavaパッケージの命名規約を標準化しています。

ルート

東海高等学校のドメイン
tokai-jh.ed.jp
をひっくり返し、ハイフンをアンダースコアに置き換えたものを東海中学校・東海高校のルートパッケージ名とします。
jp.ed.tokai_jh

マイコン班の音

このルートパッケージの下に各部活名が置かれます。
対応表はマイコン班が管理しています。
変更を希望する部活はマイコン班に申請し、許可されれば変更されます。

部内での分割

さらに電波科学研究部はマイコン班とハム班に分かれるので、その下を次のように分けます。

マイコン班micom
ハム班ham

その下

そこから先は部内で自由に管理することができます。

推奨される方法

  • 個人のプロジェクト
    • まずユニークな個人のニックネームを置く。
    • さらにプロジェクト名を置く。
    • その先は自由にサブパッケージを作る。
  • 部全体のプロジェクト
    • プロジェクト名を置く。
    • その先は自由にサブパッケージを作る。

例

電波科学研究部(radio)マイコン班(micom)のyappyのテストプロジェクト(testproject)のパッケージ
package jp.ed.tokai_jh.radio.micom.yappy.testproject;
public class TestProject {
  public static void main(String[] args){
    System.out.println("This is test.");
  }
}
